Power map generation utility

A series of Matlab functions have been developed to simulate scenarios with several sources radiating spherical waves. It can be used to create maps of amplitude / power on different setups and see how they interfere.

Developed scripts

  • generatePowerMap.m to produce the map file.
  • generateFractionalDelayIR.m to generate impulse responses used to delay signals with fractional values.
  • plotAmplitudeMap_image.m to display the map files generated.
